# Silverbullet Repository

Here are my libraries (currently just one) that I created/modified for my Silverbullet instance. Feel free to fork, modify, and share!

> Mirror Note: The repository is hosted on Codeberg [https://codeberg.org/swip3798/silverbullet-repo](https://codeberg.org/swip3798/silverbullet-repo). There's a push mirror on GitHub, but no development takes place there.

## Installation

- Open Library Manager (Command: `Library: Manager`)
- Add repository (`https://codeberg.org/swip3798/silverbullet-repo/raw/branch/main/swip3798-repo.md`)
- Add Libraries as desired

## External APIs

Some libraries utilize external APIs (OpenStreetMap, Photon) to achieve their functionality. Those APIs are free to use and without an API Key, but they all function under Terms-of-Service and Fair Use principles. Please use the Libraries responsibly and avoid overly straining their servers.

### Terms-of-Service

- [OpenStreetMap Terms of Use](https://osmfoundation.org/wiki/Terms_of_Use)
- [OpenStreetMap Copyright](https://www.openstreetmap.org/copyright)
- [Photon API](https://photon.komoot.io)
